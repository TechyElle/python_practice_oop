# Final Exam - Python OOP Scripts

This folder contains Python scripts for the OOP practical exam.

## Structure
Scripts are grouped by batch:
- `Batch_01_arithmetic_and_comparisons/`
- `Batch_02_arithmetic_and_comparisons/`
- `Batch_03_arrays_and_duplicates/`
- `Batch_04_numbers_and_duplicates/`
- `Batch_05_name_formatting/`
- `Batch_06_string_ops/`
- `Batch_07_string_ops/`

Each script follows the repo style:
- One **class**
- Exactly one **`run()`** method
- Reads from **stdin** via `open(0)`
- Prints the required output

## How to run
Run any script directly:

```bash
python final_exam/Batch_01_arithmetic_and_comparisons/prog01_bigger_number.py
```

## Quick runner (optional)
You can also run a script by passing its path to:

```bash
python final_exam/final_exam_runner.py <script_path>
```

Example:
```bash
python final_exam/final_exam_runner.py final_exam/Batch_01_arithmetic_and_comparisons/prog01_bigger_number.py
```
